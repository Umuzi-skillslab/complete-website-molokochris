## Approach to Fixing Issues

- Updating all the pages to include proper 'charset' and 'viewport' meta tags.
- Add proper semantic tags to fix the document structure, by wrapping the navigation 'li' items with anchor tags.
- Fix contact form to include 5 input types (text, email, textarea, radio, and checkbox)
- Adding correct label to the input fields
- Adding consistent 'DOCTYPE' declaration across all pages
- Adding more descriptive alt attribute text on image tags

For CSS:

- Expand the stylesheet to use 5+ selector types including element, class, ID, descendant, and pseudo-class
- Adding hover and focus semantic classes to navigation links, form inputs and the footer anchor tag
- Removing the lightblue hero background to fix the color contrast issues and adding background image instead
- Center aligning the footer text content
